# Innovatech---Retrofit-Gravadora-a-Laser-
Retrofit: Gravadora a Laser. Esse processo envolveu a substituição da controladora original por um modelo mais moderno, a troca do módulo laser por um mais potente. Além disso, foi projetada e construída uma cabine de proteção e uma bancada ajustável, com o objetivo de aumentar a segurança e ergonomia durante a operação do equipamento.
